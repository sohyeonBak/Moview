## 프로젝트 소개 💡

영화 리뷰 SNS로 개인계정으로 영화 리뷰 작성, 공감/비공감, 댓글쓰기 같은 기능으로 영화 내용을 소통할 수 있는 페이지
</br>


## Build&Start 🏷

Frontend Build
```
npm run build
```
Frontend Start
```
npx pm2 start npm -- start
```
Backend start
```
sudo npm start app.js
```
